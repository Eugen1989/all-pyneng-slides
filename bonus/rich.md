## Модуль Rich

---

### Rich

Rich позволяет создавать красивые приложения в CLI.

Возможности Rich

* rich print
* более понятные traceback
* inspect
* добавление цвета/стиля при выводе в терминал

* Handler для logging
* prompt
* progress bar
* spinner
* вывод красивых таблиц
* отображение markdown
* tree
